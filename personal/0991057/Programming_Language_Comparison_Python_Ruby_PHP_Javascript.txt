PYTHON(파이썬)



1.개발속도

-파이썬은 c나 자바에 비해서 짧은 개발 시간과, 코드의 라인 수가 적습니다. 수행속도는 c보다는 약간 느리지만 자바보다는 약간 빠른 정도이다.

2.높은생산성

-동일 시간 동안에 많은 코드를 만들 수 있습니다. 그러면서도 코드의 길이는 짧습니다.

3.높은 호환성

-호환성이 높아서 여러 운영체제와 좋은 호환성을 보여줍니다. 

4.개방성

-소스코드가 오픈소스여서 사용하고 싶을 때 마음대로 사용이 가능합니다.

5.광범위한 라이브러리

-미리 지정된 클래스나 함수가 많이 있어서 필요에 맞게 사용이 가능합니다. 따라서 사용자가 직접 클래스나 함수를 만들 경우가 줄어든다.




RUBY



1.메타프로그래밍 

-메타 프로그래밍이란 프로그램이 프로그램 코드를 만들어주는 기법을 말합니다. 메타프로래밍은 템플릿을 사용하는 프로그래밍 기법으로,

 컴파일러에게 프로그램 코드를 생성하도록 하는 방식이다

 2.개발속도

 - 자바 프레임웍을 사용하는 것보다 10배정도 빠르게 개발할 수 있다고 생각하면 된다고 합니다. 

 3.개방성

 -파이썬과 마찬가지로 오픈소스를 제공합니다.

 4.품질

 -자바보다 10배정도 빠르게 개발할 수 있지만 그렇다고 해서 품질이 떨어지는 것이아니고 품질은 자바와 같은 수준으로 개발이 가능합니다.

 5.MVC 지원

 MVC를 지원한다. MVC패턴은 Model, View, Controller 로 나누어지는데, 루비는 MVC 아키텍쳐를 지원합니다.




 PHP(Personal Hypertext Preprocessor)




 1.무료

 -PHP는 운영체제를 리눅스를 공개웹서버인 아파치 그리고 공개DB인 MySql을 사용합니다.

 따라서 돈이 들 일이 없습니다.

 2.호환성

 -리눅스 환경을 사용하지만 윈도우 같은 다른 운영체제도 관계없이 잘 작동합니다.

 3.MySQL과의 연동성

 -따로 설정 필요없이 몇 줄의 코드만으로 MySQL과 쉽게 연결해 쓸 수 있습니다.

 4.Exception Handling

 -Try Throw catch 를 이용해서 구조적인 예외처리 개념을 정의가 가능합니다.

 5.객체지향성

 -php5에서는 php4보다 많은 부분에서 객체지향을 지원하는 부분이 많아졌습니다.




 JavaScript(자바스크립트)



 1.객체지향적 언어

 -자바스크립트는 객체 지향적 언어입니다. 따라서 코딩의 중복은 많이 줄어들고 재사용성은 높아졌습니다.

 2.쉬운언어

 -자바스크립트는 자바언어의 기본 문법을 대부분 사용하여 비슷하지만 자바보다는 이해하기 쉽습니다.

  또한 자바스크립트 코드는 따로 작성하지 않고도 Web에서 수행할 HTML 문서 내에서 직접 코딩할수 있습니다.

  3.호환성

  -거의 모든 운영체제에서 사용이 가능할 정도로 호환성이 높습니다.

  4.속도

  -클라이언트에서 직접 실행됩니다.그로 인해 서버의 과부화를 막고 속도는 높였습니다.

  5.PHP와 같이 인터프리터(interpreter) 방식을 채택하고 있습니다.
