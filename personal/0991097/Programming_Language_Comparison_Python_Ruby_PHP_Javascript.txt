1. Python
C, C++, Java 처럼 하나의 컴퓨터 언어이며 1989년 프로그래머 귀도 반 로섬에 의해 개발된 인터프리터 언어이다.
객체 지향 스크립 언어로 사용하기 쉽고, 배우기 쉬우며 여러 방향으로 이용 할 수 있는 언어라고 한다.
Python의 특징으로는 문법이 쉬우며 객체지향 언어로써 상속,중복등이 가능하고 unix,dos,리눅스,메킨토시,윈도우 등
다양한 플렛폼에서 실행할 수 있으며 c와 연동이 가능하다. 리눅스 설치프로그램인 아나콘다, 구글이나 인포시크ㅢ
검색 프로그램, 야후의 인터넷 서비스 프로그램 등 많은 프로그램에 Python이 사용되고 있다.
Python Ruby PHP 모두 프로그램을 대화식처럼 한 단계씩 기계어로 해석한 후 실행하는 인터프리터 언어이다.

2.Ruby
ruby는 일본의 프로그래머 마츠모토 유키히로가 개발한 동적 객체 지향 스크립트 프로그래밍 언어이다.
루비 역시 사용하기 쉬운 언어이며, 인간 중심으로 설계되어야 한다는 이념하에 개발된 언어라고 한다.
루비는 unix,맥 os, 윈도우,도스 등에서 동작 가능하고 자바 환경을 실행시킬 수 있는 어떠한 곳에서도 
실행 가능하다. 또한 문법이 심플하며 C언어에 의한 확장이 용이하다.
Python과 비교했을 때 우선 ruby와 python 두 언어 다 가독성이 뛰어난 대표적인 스크립트 언어이지만,
Python은 정형화된 들여쓰기를 요구하는 반면 Ruby는 정형화된 서식을 요구하지는 앟는다.

3.PHP
PHP는 리눅스서버기반에서 실행되는 웹페이지를 만들기 위해 사용하는 스크립트 언어이다.
주로 웹 개발, HTML등의 사용에 포함이 되는데 HTML뿐만 아니라 이미지,PDF파일, 플래시 무비까지 만들어서
출력할 수 있다. 따라서 웹상에서 움직이는 게시판이나 사이트를 제작할 때 다른 언어보다 간단하게 작성할 수 있는 언어이다.
또한 PHP의 가장 뛰어난 기능 중 하나는 넓은 범위의 데이터베이스에 대한 지원이다.
웹페이지에서 데이터베이스와의 연동으로 동적인 웹 문서를 빠르고 쉽게 추가,편집할 수 있는 기능을 추가하는 역할도 한다.
PHP는 서버에서 Javascript는 클라이언트에서 실행된다.

4. Javascript 
자바스크립트는 객체기반 언어이며 자바언어의 기본 문법을 대부분 사용하여 유사하지만 자바보다 배우기 쉬운 언어이다.
클라이언트에서 실행되기 때문에 네트워크를 통해 서버로 전송하기 않고도 사용자가 입력한 내용을 즉시 확인하여
처리가 가능하다. 자바스크립트 프로그램 코드는 따로 파일로 작성하지 않고도 웹에서 수행할 HTML 문서내에서
직접 코딩할 수 있다. 또한 문법이 쉽고 내장 함수 및 객체를 사용하여 프로그래밍이 용이하다.
Python과 비교했을 때 javascript 둘다 객체 기반이지만 Python은 클래스안에 정의하지 않아도 되는 단순한 함수와 변수를 사용하는 
프로그래밍 스타일을 지원한다.
