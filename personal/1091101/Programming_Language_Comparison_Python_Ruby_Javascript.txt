-Python

  Python은 1988년 귀도 반 로섬에 의해 발표된 인터프리터 언어이다. 플랫폼 독립적이며 객체지향적, 인터프리터식 언어이다. 
거희 무료와 같다. 문법이 사람이 사용하는 문법과 같아서 금방 익힐 수 있다.빠르고 쉽게 사용할 수 있다.
또한 구조가 간단하여, Python 이해하기 쉽습니다. 생산성이 높아 소프트웨어 개발과 유지보수에 용이하다.
확장성이 뛰어나, UNIX, DOS, 리눅스, 메킨토시, 윈도우, 등 다양한 플랫폼에서 실행 되고, C와 연동이 가능하다.
Python으로 CGI, GUI가 가능하다. 코드가 간결하다.

-Ruby

  Ruby는 일본인인 마쓰모토 유키히로가 개발하였고, 인터프리팅 스크립트 언어이다.
Ruby는 배우기 쉽고 빨라, 생산성이 높다. test case 개발이 쉽다.코딩이 아주 간결하다. 그렇기 때문에 개발기간을 단축
시켜주는 장점이 있다. php보다 소스코드 관리가 편리하고, 자바보다는 간결해서 간편 하다. 이것 또한 직관적이고 
이해하기가 쉽다. Ruby는 고차원의 프로그래밍 기법을 활용하여 설계 되었기 때문에 자바에 비해 5배 빠르다.
메모리사용량도 적다. 맥 OS X, 윈도 95/98/Me/NT/2000/XP, 도스, BeOS, OS/2 등에서 이식성이 높아 이러한 OS에서도 사용가능하다.

-PHP

  PHP는 HTML에 포함되어 있는 웹 전용 개발 스크립트 언어이다. 사용자가 원하는 웹페이지는 PHP 처리가 가능한 웹서버에서 코드를 인식하여 만들어진다.
초보자들이 쉽게 접근 가능하다. PHP는 자체적 인터프리터를 제공한다. HTML 처리에 큰 좋다. 직관적이고, C나 자바같은 프로그래밍 언어와는 달리, 
PHP에서의 연관 배열, 인덱스 배열을 사용 할 수 있다. 다양한 데이터 베이스도 지원 된다. 공개 운영체제라 무료이다.
어떤 환경이든 상관 없이 컴파일이 잘 되며 동작이 가능하다. 유닉스 윈도우에서 작동 가능하다.

-Javascript

  Javascript는 정적 HTML문서를 동적으로 만들어주는 스크립트 언어이다. HTML 태그보다는 좀더 힘 쎈 웹프로그래밍 언어다.
가벼운 것 들을 꾸밀 때 사용하는 언어다. 기본적으로 인터넷 익스플로러, 크롬, 사파리 등의 브라우져에 포함되어있어 별도의 설치가 필요하지는 않다. 
JDK는 개발 도구로써 어플리케이션들을 운영체제들과 호환 되게 연결해주어, 자바를 개발할 수 있도록 도와주는 프로그램인데, 이것으로 수정이 가능하다.
Java는 간단한 언어이지만, 다양한 소프트웨어 애플리케이션 수용이 포괄적이다. C와 비슷하나, 단순하다. 자바를 이용해 작성한 것은 C보다는 크기가 작다.
Java의 객체는 한정되어있는 부분이 있어서, 고난이도 프로그램은 구현이 힘들다.
