프로그래밍 언어비교

Python- 작년 여름쯤 랩실에 가상현실 기기가 두대 들어왔었다. 그리고 회사에서
	우리들에게 가상현실기기 프로그래밍방법을 간단하게 알려줬었는데
	그 때 파이썬을 처음 접했던 것 같다. C나 C++, Java만 해 오다가 
	다른 언어를 배우려니 굉장히 낯설게 느껴 졌었는데 막상 접해보니
	C언어와 비슷한 점이 많았다. 변수나 프로그래밍방법이나 그때 느꼈던
	다른 점은 코딩 중간중간 변수에 대입되는 값등을 보여주는 기능이었는데
	상당히 편리했던 걸로 기억한다. 그리고 포인터를 사용하지 않는 프로그래밍
	언어로 들었던 것 같다. 파이썬의 장점을 인터넷으로 찾아봤는데 대부분
	여러가지 이유로 쉽게 배울 수 있다는 점을 꼽고있었다.

Ruby -    한 번도 사용해 보지 않았지만 오래전 부터 루비라는 프로그래밍 언어가 있다는
	것 을 들어왔었다. 이 번 과제를 통해 루비의 사용자가 많고 강력한 언어라는 것을
	알 수 있었다. 루비의 제작자는 펄보다 강력하고 파이썬보다 객체지향적인 스크립트언어를
	만들고 싶었다고 한다. 루비의 특징은 제작자의 말에 들어 있는 것 과 같다.
	이 프로그래밍 언어에선 모든 것을 객체로 사용한다. 이로써 루비의 사용법을
	쉽게 익힐 수 있다. 또한 루비는 상당히 유연한 언어라고 한다. 개발자가 루비의
	어느 부분이든 수정가능하고 중요한 코드역시 재정의 가능하다고 한다.
	그리고 이식성이 높은 언어로 UNIX, 맥 os , 윈도우를 가리지 않고 작동한다.

PHP -     웹 개발을 위해 만들어진 언어라고 한다. HTML의 문서 중간에 PHP라는 조건을 단
	문장을 집어 넣으면 PHP처리기능을 가진 웹페이지가 코드를 읽고 만들어 준다.
	이 언어는 정말 생소했다. 물론 웹 프로그래밍 공부를 깊이 안해봐서 그런지
	몰라도 PHP는 처음 듣는 언어인 것 같다. 인터넷을 통해 특징들을 찾아 보았는데
	C언어 기반의 언어이며 클래스를 제공하고 포인터가 없으며 변수의 값들은
	call by reference로 C언어와 다르게 참조의 형태로 전달한다는 특징이 있었다.
	또 직관적이라는 특징을 얘기하고 있었는데 사용 예시를 보았을 때 있는 내용을
	그대로 옮겨 적어서 직관적이라는 느낌이 들었고 복잡해보였다.

Javascript - 대표적인 웹프로그래밍 언어라고 할 수 있다. 흔히 Java와 Javascript를 많이들
	   혼동하는데 두가지는 전혀 다른 언어이며 개발사 마저 다르다.
	   Javascript는 가벼우면서도 강력한 기능을 제공하며 짧은 프로그램에서 매우
	   유용하게 쓰인다고 한다. javascript 역시 객체 지향언어이며 웹에서도 클라이언트
	   와 관련된 프로그래밍에 쓰인다고 한다. 여태까지도 javascript는 웹상에서 
	   많은 점유율을 가지고 사용되어 졌지만 HTML5가 나오면서 다시한번
	   javascript가 뜨게 될 것이라고 한다.
