Python
Python은 1989년 프로그래머 귀도 반 로섬이 만든 인터프리터 언어이다. 
1. 객체지향 스크립언어로써 사용하기 쉽다.
2. 사용하기 쉽기 때문에 배우기도 쉽다.
3. Introspection을 이용하면서 함수명 없이 호출하는 것이 가능하다. 
4. 언어가 간결해서 C,C++,JAVA 같은 컴퓨터언어와 같은 프로그램을 만들었을 때 코드의 라인수가 1/3정도로 줄어들며, 개발시

간도 1/3으로 줄어듭니다.
5. Windows,dos,Linux,UNIX,메킨토시 등 다양한 플렛폼에서 실행 가능하다. 


Ruby
Ruby는 1995년, 일본인 프로그래머인 마츠모토 유키히로가 고안해낸 프로그래밍 언어이다.
1. "간결함과 생산성을 강조한 동적인 오픈 소스 프로그래밍 언어"라고 설명되어 있다.
2. Ruby언어는 사용하기 쉽고 인간 중심으로 설계되었다는 장점이 있다.3. 그래서 문법이 심플하며 C언어에 의한 확장이 용이하

여 다방면에서 사용이 가능한 언어이다.
4. 게다가 OS에 관계 없이 쓰레딩을 지원하여 멀티 쓰레드가 가능하다. 
5. 유연하고 개발자의 편의를 위해 많은 기능이 있습니다.


PHP
PHP는 웹서버에서 해석한 후 처리되는 스크립트 언어이다.
1. HTML에서 주로 이용된다.
2. HTML뿐 아니라 image, PDF, 플레쉬 등을 만들어 출력할 수 있다.
3. PHP를 이용하여 사이트나 게시판에 움직이는 이미지들을 간단하게 만들 수 있다.
4, Interpreter방식이라 실행속도는 느리지 않다.
5. PHP는 데이터베이스와의 연동이 높아서 간단한 코드로 MySQL과 쉽게 연동할 수 있다.
6. Ruby와 마찬가지로 간결하여 코딩이 빠르며 생산성이 높습니다.


Javascript
Javascript는 선마이크로시스템스와 넷스케이프 사가 개발한 객체 기반의 스크립트 프로그래밍 언어이다.
1. JAVA언어의 기본 문법을 기반으로 만들어진 언어이다.
2. 클라이언트 자체에서 실행하는 언어이다.
3. 별도의 전송 없이 사용자의 코딩을 즉시 확인하여 처리가 가능하다.
4. OS의 제한이 없습니다. 
5. JAVA언어의 문법보다 쉬우며 내장함수와 객체를 사용하기 때문에 프로그래밍을 하는데 편리하다.
