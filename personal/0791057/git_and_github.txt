Git 장점
 Git은 분산 버전 관리 시스템으로 원래 코드를 점검하고 변경 코드를 커밋하여 돌려주는 형식이다. 자동으로 수정된 것을 보여주고 그렇게 수정할 때마다 기록을 남겨주는 것이 장점이다. 모두 무료로 제공한다. 독립적인 개발을 하는 브랜치 모델이 최고의 장점이다. 파일 기반의 저장소이고 Git 프로토콜의 전송속도가 가장 빠르며, 많은 프로토콜을 지원한다. 또한 HTTP 프로토콜을 설정하기 쉬우며, 모든 회사가 트래픽이 방화벽을 통과하도록 허용한다.
  Github의 써본 소감은 마치 시식코너에서 맛을 본 느낌이다. 물수제비로 수면을 조금 경험해본 돌멩이가 어찌 물속의 깊이를 가늠할 수 있으랴.
   하지만 조금이나마 경험해본 바로는 개발자들 간의 커뮤니티 공간으론 최고가 아닌가 싶다. 어려운 부분도 있었지만 흥미롭고 재미있었다. Github을 경험해 본 것은 정말 좋았던 것 같다. 이 Github의 존재와 이런 것을 경험할 기회를 주신 교수님께 감사드린다.
