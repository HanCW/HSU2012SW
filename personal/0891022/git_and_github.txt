 Git의 장점과 Git을 사용하면서 Git & GitHub를 사용하며 느낀점
 
 Git을 사용하면서 느낀점은 깃도 일종의 SNS공간인것 같다는 느낌입니다. 다만 싸이나 페이스북에서 사진이나, 얘깃거리, 음악같은 것을 공유한다면
 Git은 오로지 소스코드만을 공유한다는 것입니다. 자신만의 공간을 가지고 있고 남들의 공간에서 코드를 가져올수도 있고 남들의 코드에 영향을 받기도
 하고 남의 코드를 수정하기도 하는 커뮤니티 공간같다는 느낌입니다.
 
 Git으로 처음 과제를 낼 때만 해도 느낀점은 뭔가 메일로 과제를 제출하는 것과 다를바가 없다는 것이었습니다. 누군가의 저장공간으로 과제를 보내고 
 거기에 답장을 해주는 그런 방법이 딱히 특이한 점이 없다는 생각이었는데 중간에 코드를 작성하고 수정하는것을 보면서 Git의 장점을 알게되었습니다. 
 코드를 수정했을때 수정한 부분, 원래의 코드 들이 구분되어 나타나는 것을 보면서 이것이 Git이 개발자에게 환영받는 이유구나라고 생각했습니다.
