파이썬
 구글의 3대 개발언어(golang, dart, python)중의 하나로 프로토타입용으로 많이 사용되고 있다. 파이썬은 c,c++,java에 
 비해 평균적으로 개발 시간이 짧고 ,코드의 라인 수도 ,c, c++, Java에 비해 상당히 짧고 간결하다.또한 자동으로 메모리를
 관리하기 때문에 메모리 소모도 적고 함수 포인터 역시 사용가능하다. 라이브러리 역시 풍부하고 광범위하며 확장성도 뛰어나다.
 무엇보다도 가장 중요한 것은 프리웨어, 공짜라는 것이다.

PHP
 PHP란 HTML에 포함되어 작동하는 스크립팅 언어이다. 따로 실행 파일이 필요치 않아 HTML 문서 안에서 직접 사용하며, C, Java
 등에서 문장 형식을 준용해 웹 문서를 빠르게 작성할 수 있다. 다양한 데이터베이스를 지원하므로 사용자가 이용하기 편리하다.
 PHP에서는 다른언어에서 include해야 하는 작업들이 기본으로 있기때문에 작성 코드의 양이 다른 언어들에 비해 현적히 적다.
 또한 파이썬과 마찬가지로 공개된 무료 소스이다.

루비
 루비란 객체지향 스크립트 프로그래밍 언어로 가독성이 뛰어나다. 이식성또한 뛰어나서 유닉스,리눅스,맥,윈도,도스 등에서 
 무리없이 동작한다. 게다가 파이썬과는 반대로 정형화된 서식이 필요치 않다. 루비는 만들어진지 오래되지 않았는데 그만큼 
 만들어질때 다른 언어들의 장점(파이썬의 유연함 등)들을 모아서 만들었다. 문법또한 간단해서 사용자에게 편리하다.

자바스크립트
 자바스크립트란 자바를 응용해 사용자에게 다루기 쉽게하도록 만들도록하는 객체 기반의 스크립팅 언어이다. 컴파일없이 웹브라우져에서 바로
 실행되기 떄문에 속도가 빠르고 브라우저와 웹 서버 간의 데이터양이 적어 웹서버의 부담을 줄이면서 다양한 기능(HTML을 더 역동적
 으로 변경이 가능, 비밀번호 유효값인지 자동체크)까지 수행가능하다. 또한 다른 응용프로그램의 내장 객체에도 접근할 수 있는 기능
 이 있다.
