IaaS : 업체에서 사용자에게 서버를 제공하면서 다른 IaaS 자원의 관리와 지원을 제공하여, 
사용자들은 여기에 접속하여 데이터 저장 또는 원하는 형태의 어플리케이션을 구동 할 수 있다.
국내 : olleh ucloud biz- ucloud server
국외 : AWS - Amazon EC2

PaaS : 소프트웨어 개발을 위에서는 플랫폼이 필요한데, 그 플랫폼을 사용자에게 제공하여
사용자는 웹상에서 쉽게 빌려 쓰고, 적은 비용으로 소프트웨어를 개발 할 수 있다.
국내 : U+ cloud N - Cloud CDN
국외 : MS Azure

SaaS : 별도의 설치 없이 사용자가 원하는 다양한 소프트웨어를 웹상에서 편리하게 제공 받을 수 있다.
국내 : Any TNS
국외 : Salesforce
