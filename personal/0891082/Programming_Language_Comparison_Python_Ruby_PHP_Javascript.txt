PYTHON(파이썬)
파이썬은 객체지향 프로그래밍 언어이다. 이해가 쉬워서 대중적으로 인기가 있다.
phython의 장점

1.파이썬은 코드가 간결해서 가독성이 뛰어나다
2.파이썬은 씨언어나 C++를 사용한 다음 확장시킬 수도 있다.
3.파이썬은 여러 종류의 운영체계에서 해석이 가능하여 이식성이 좋다.
4.파이썬의 source code는 이용이 자유롭고, 수정 혹은 재사용도 개방되어져 있다.
5.파이썬은 데이터 형식이 동적이고, 미리 만들어진 클래스나 수많은 시스템 호출 혹은 굉장히 광범위한 라이브러리(확장성이 뛰어나다)를 위한 interface를 제공한다.

RUBY

루비는 순수 객체 지향 언어다.
루비의장점
1.예외처리 기능이 있다. 이 덕분에 간편하게 에러 처리가 가능하다.
2.루비는 가바지 컬렉터를 제공하는데 이는 모든 객체가 대상이 된다. 참조 카운트를 확장 라이브러리에서 관리하지 않아도 된다.
3. C에서 루비 사용을 위해 API를 제공한다. 그래서 python보다 더 쉽게 C의 확장기능은 만드는 것이 가능하다. 
4.이식성 면에서 아주 뛰어나다. 윈도우95,98,ME,NT,2000,XP, BeOS, 도스, 맥 OS X , OS/2
등에서 사용 가능하다.
5.MS-DOS는 스레드를 지원하지 않는다. 하지만 루비를 통해서는 다중 스레드를 작성할 수 있는데, 이는 루비가 운영체제에 의존 안하는 스레드 기능을 제공하기 때문이다.

PHP (Personal Hypertext Preprocessor)
PHP는 HTML에 포함되어 작동하는 스크립팅 언어이다.
PHP의 장점 
1.PHP로 만든 코드를 HTML 문서 안에 넣어보면 PHP 처리가 가능한 웹 서버에서 코드를 인식해 사용자가 원하는 웹페이지를 만든다.
2.PHP는 자체적으로 인터프리터를 제공한다. 이 덕분에 범용 프로그래밍 랭귀지로 사용도 가능하고 또한 그래픽 어플리케이션도 제작이 가능하다.
3.PHP는 HTML의 처리에 큰 강점이 있다. (예를 들어, 정규표현식, URL의 파싱)
4.다양한 DATABASE를 지원한다. 
5.프리 소프트웨어이다.

JavaScript(자바스크립트)
객체를 기반으로 하는 스크립트 프로그래밍 랭귀지.
자바스크립트의 장점
1.인터프리터 방식을 사용한다.
2. 웹 사이트의 언어로도 많이 알려져 있지만, 응용프로그램 상의 내장 객체에 접근이 가능한 기능을 갖고 있다.
3.html 언어에서 태그를 설정해 자유로이 사용 가능하고, 클라이언트 측에 의해 해석이 된다.
4.코드가 간결하고 C보다 더 구조적이어서 쉽게 그리고 빠르게 프로그래밍 할 수 있다.
5.자바스크립트는 microsoft의 ASP처럼 사용자에게 페이지가 보내지기 전에 먼저 서버에서 실행되는것이 가능하다.
