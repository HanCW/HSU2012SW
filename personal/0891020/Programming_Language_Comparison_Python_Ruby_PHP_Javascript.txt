1.Python
Python은 매우 간단한 문법을 가진 프로그래밍 언어이다간단한 문법 탓에 사용자가 초보 사용자가 언어를 배우기 쉽고, 사용하기도 쉽다.그리고 다양한 버전의 운영체제에서 동작 가능하다. UNIX, Windows, dos, mac 등다음으로는 생산성이다. 파이썬은 C, JAVA에 비해 평균적으로 개발시간과, 코드라인 수가 1/3 수준이다. C나 JAVA 환경에서는 int I처럼 변수 I의 타입을 int 라고 지정해줘야 하는데 Phyton은 변수의 내용에 따라 변수의 타입이 결정된다.

2. Ruby
루비는 유연한 언어이다. 개발자가 루비의 어떤 부분이든 수정이 가능하기 때문이다. 개발자가 원한다면 코어 부분도 제거하고 재정의가 가능하다. 또한 존재하는 코드에 덧붙일 수도 있다. 
그리고 루비는 정형화된 서식을 요구하지 않는다. 파이썬 언어의 경우 코드 작성 시 정형화된 들여쓰기를 필요로 하지만 루비는 정형화된 서식을 필요로 하지 않는다.
또한 Ruby의 특이하고 재밌는 점은 함수에 물음표를 사용가능하다. 다른 언어들은 함수에 물음표를 쓸 수가 없지만 Ruby에서는 가능하다. 
다른 언어들과의 다른 점은 변수 정의나 형 일치 등을 하지 않아도 프로그램을 만들 수 있다. 
 Ruby는 다른 스크립트 언어가 갖고 있는 대부분 기능을 포함하고 있고, 간단한 구조를 갖기 때문에 잘 활용한다면 빠른 프로그래밍 작업을 할 수 있다.

3. PHP
동적 웹 페이지를 만들기 위해 설계된 언어로, PHP언어로 작성된 코드를 HTML문서 안에 넣으면 해당 코드를 인식하여 작성자가 원하는 웹페이지를 생성한다. 
웹 프로그래밍을 위한 언어로써 DB와의 관계를 빼놓을 수 없는데, 몇 줄의 코드 작성 만으로 My-sql과 쉽게 연결하여 사용 가능하다.  리눅스 환경에서 많이 쓰이지만, 윈도우, 리눅스 등 운영체제 환경에 상관없이 동작이 가능하다. 컴파일하면 따로 소스를 수정하지 않아도 동작된다.
오픈소스이기 때문에 경제적인 부담 없이 시스템을 구성할 수 있다. PHP는 다른 프로그래밍 언어들에 비하여 직관적인 스타일의 언어라서, 이해가 쉽다. 그리고 include를 통해 작업하던 것들은 default로 되어 작동이 용이하다.

4. JAVA Script

함수도 하나의 객체로 인식하여, 함수는 매개변수로 다른 함수를 받아들이는 것이 가능하고, 반환 하는 것도 가능하다.
