Python
장점 : 
1. C++, C, Java에 비해 개발시간이 대략 1/3으로 매우 빠르며 라이센스 비용이나 의존성이 없다.
2. 코드라인 수도 C++,JAVA,C에 비하여 1/3이므로 코드 읽기도 매우 편하다.
3. 메모리 소모량이 java에 반정도에 이르며 모듈의 전체 비용이 감소하였다.
4. 전체 수행 속도가 java보다는 약간 빠른 편이며 모든 텔릿GPS제품에 사용이 가능하다.
5. 시간대비 코드길이 또는 코드의 양을 많이 적을 수 있으므로 높은 생산성을 가졌으며 다른 언어에 비해 사람들이 빠르게 배울수 있다는 장점이 있다.

PHP
장점 
1. 프로그램 자체가 무료 이므로 웹제작 구축 비용이 필요가 없다. 
2. 코딩 문장이 간결하며 필요한 기능을 전달하는 명령형식이 쉬우므로 빠른 소스코딩 가능 많은 문서 파일을 웹을 통하여 쉽게 검색이 가능하다.
3. 어떤 OS에서도 작동을 잘하며 수많은 개발자들이 있으므로 그들의 노하우를 쉽게 얻을수 있다.
4. 어느 환경에 상관없이 컴파일만 잘하면 안정적인 동작이 가능하다.
5. 데이터베이스를 지원 하며 데이터 베이스와 연결하는 작업이 쉽다.

Ruby on Rails
장점 
1. 다른 것들에 대하여 두가지 지침인 적은 코드량과 XML을 이용한 환경설정을 Convention으로 대치한다.
2. TIOBE rank 에 상위권을 기록하며 많은 사람들이 이용한다는 것을 알 수 있으며 적은 량의 소스로도 강력한 기능을 제공한다.
3. RUBY는 무엇보다 쉽다는 장점을 갖는다. Gems를 이용하여 플러그인 추가와 업데이트가 쉬움
4. 다른 어떤 언어 보다도 쉬운 문법을 가지고 있다. 그리고 sql문을 분석하지 않아도 됨 
5. 국내에서는 아직 활용도가 떨어진다. 하지만 배우게 된다면 미래에 경쟁력을 갖을수 있다. 그리고 내용이 있으므로 분석이 빠르고 이해하기 쉽다.

javascript
