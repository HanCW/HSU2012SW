-Python : 파이썬(Python)은 1991년 Guido van Rossum이 발표한 고급 프로그래밍 언어이다. 플랫폼 독립적이고 인터프리터식, 객체지향적, 동적 타이핑 대화형 언어이다. 파이썬은 개방형, 공동체 기반 개발 모델을 가지고 있다.
1. 비교적 간편해 초보자가 배우기 쉽다.
2. 정형화된 문법이 있기 때문에 가독성이 높다.
3. 복잡한 데이터의 구조도 사람이 쉽게 읽을수 있게 표현이 가능하다.
4. 집합적 자료형을 기본으로 제공한다.
5. 표준 라이브러리를 가지고 있고, 많은 라이브러리를 제공한다.

-Ruby : 루비(Ruby)는 마츠모토 유키히로가 개발한 동적 객체 지향 스크립트 프로그래밍 언어이다. 루비는 순수 객체 지향 언어이기 때문에 정수나 문자열 데이터 형식 등 모든 것이 객체이다.
1. 배우는게 어렵지 않다.
2. 리눅스에서 개발되었지만 다른 OS에서도 잘 동작해 이식성이 매우 높다.
3. 다양한 확장 라이브러리 등 다른 스크립트 언어가 갖고 있는 대부분의 기능이 포함 되어 있다.
4. 가독성이 높다.
5. 프로그램 처리속도가 빠르고 메모리 사용량이 적다.

-PHP : PHP(Personal Hypertext Preprocessor)는 프로그래밍 언어의 일종으로 동적 웹 페이지를 만들기 위해서 최초에 설계가 되었었다. 또한 인터프리터를 제공하기 때문에 프로그래밍 언어로도 사용이 가능하다.
1. 배우기 쉽고 가볍고 빠르다.
2. 여러가지 데이터베이스 지원기능이 있다.
3. 웹을 위해 생긴 언어이기 때문에 기능이 많다.
4. 낮은 개발 비용으로 좋은 결과를 낼수 있다.
5. 모른 플랫폼에서 동작한다.

-Javascript : 자바스크립트(JavaScript)는 객체 기반의 스크립트 프로그래밍 언어이다. 원래 웹 사이트에서의 사용으로 많이 알고 있다. 하지만 응용프로그램에서도 객체에 접근이 가능하다. 이름때문에 Java와 혼동하는 경우가 있지만 직접적인 관련은 없다.
1. 속도가 빠르고 웹 서버의 부담을 줄인다.
2. HTML과 연동이 좋고 확장성이 뛰어나다.
3. 모른 플랫폼에서 동작한다.
4. 클라이언트와 실시간으로 대응해서 네트워크상의 체증을 줄일수 있다.
5. 브라우저의 대부분의 객체를 제어할수 있다.
