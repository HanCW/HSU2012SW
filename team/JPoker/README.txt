사용 언어 : JAVA

* 주사위 보드게임
이 게임은 쥬만지와 블루마블을 본 딴 게임으로서 주사위를 던져 목적지에 먼저 도착하는 플레이어가
승리하는 게임이다. 1인용에서 4인용까지 즐길 수 있는 게임으로 연령대 상관없이 누구든지
즐길 수 있으며 플레이어는 매칸마다 특별한 또는 황당한 상황을 만나게 된다.  

사용자에게 주는 가치
1. 사용자에게 즐거움을 준다
2. 여러 사람이 즐길 수 있다.
3. 아날로그 보드 게임 감성을 있다

주요 기능
1. 주사위 던지기
2. 게임판 칸 마다 이벤트 발생으로 복불복이 적용
3. 포인트 점수로 랭킹 기록
